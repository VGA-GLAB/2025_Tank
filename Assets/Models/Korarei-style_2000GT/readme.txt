# Korarei-style 2000GT

この3Dモデルはトヨタ自動車工業とヤマハ発動機の共同開発によって生まれたスポーツカーである2000GT前期型のファンアートである．

## 配布内容

### 3Dデータ

- pmxモデル (MMDモデル)

- fbxモデル


### テクスチャ

本モデルはbody1，body2，tireの3つのテクスチャを使用する．それぞれに対して以下の7種類のテクスチャが存在する．

- BaseColor (ARGB)

- Roughness

- Metallic

- Normal (OpenGL (BlenderやUnityなど) およびDirectX (UEやMMDなど))

- BaseColorRGB (BaseColorのアルファなし版)

- Hight (Displacement) (Blenderではハイポリモデルに対してディスプレイス設定を行う)

- Opacity (BaseColorのアルファ要素と同じ)

- ORM (RGBにそれぞれAO，Roughness，Metallicの情報を入れたもの)

Blenderで使用する場合は`textures`フォルダ直下にあるテクスチャのみ使用する．`other`フォルダ内にあるものは必要に応じて使用できる．例えば，MMDやUnreal Engineを使用する場合はNormalテクスチャをDirectX用に差し替える．


### 作業データ

- blendデータ (Blender 4.2.6 LSTのプロジェクトファイル)

全てのライトが点灯している．Rootボーンを前後に動かすとタイヤも回転する．また，上下に少しだけ動かしただけではタイヤは地面を離れない．個別に動かしたい場合は個別にボーンを動かす．Handleで前輪を左右に動かすことができる．Retractable Headlightでリトラをパカパカできる．


## マテリアル

各使用環境で調整すること．特にpmxに関してはBaseColorのみ適用しているため注意が必要である．本モデルにはMME用のエフェクトは付属していない．

- bulbとついているマテリアルは電球となる．発光させることでライトを点灯させた様子を再現できる．

- lensやcoverとついたマテリアルは透明のカバーである．適切な透過設定を行うこと．
